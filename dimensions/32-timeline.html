本附录是 26_历史纪元.md 的精细化伴生卷。 26 号文档负责"为什么"——为什么三纪如此排列，每一纪的命题为何。 本附录负责"什么时候"——把同一段文明史按时间轴重新铺开，让每一年、每一桩私人时刻、每一次文化反应都各归其位。

凡例 · 这份年表怎么读

  • 26_历史纪元.md 已写过总览。本附录在它之上提供更精细的逐年/逐事件展开。
  • 所有日期都用相对纪年法，EOOVE 设定刻意拒绝绝对公元年——这是世界观的硬规则，不是疏漏。
  • 每一段年表都按四列推进：纪元 / 大事件 / 私人时刻 / 文化反应。
  • "私人时刻"指那些在官方史书里几乎被忽略、却在 EOOVE 内部叙事里承担情感锚点的小事件。
  • "文化反应"指公众情绪、媒体话术、街头传说、艺术作品的同步流变。

一、年代标定法

EOOVE 历史共用三套纪年体系，互相换算如下：

主纪年法 · 归档纪元（AY / After Yearzero）

  • AY 0 = 静夜事故那一年 = 零号塔启用、Morina 与死亡协议双生奇点诞生那一年
  • 此为全文明公认的"时间零点"
  • 用于一切官方文书、慢明协议条款编号、续生授权登记、熵海法庭判例索引

副纪年法一 · 探心纪（BY / Before Yearzero）

  • 所有发生在静夜事故之前的可考事件，向前数为 BY
  • BY 1 = AY 0 的前一年
  • 因为静默铺路期被官方刻意模糊，BY 数值越大，准确性越低
  • 探心纪学者通行做法：BY 30 之前的事件统一标"BY ~约值"

副纪年法二 · 续生纪元（SY / Slowlight Year）

  • SY 0 = 慢明协议正式生效那一年 = AY 19
  • 续生授权登记、续生者身份编号、慢明协议委员会判例皆按 SY 编号
  • 当下时间：约 SY 80 上下浮动（不同档案口径差 ±2 年）

AY 18 · 熵海法庭年

大事件Morina 的私人时刻文化反应
Morina 在熵海见到三类副本——执着说"我就是原来的我"的、彻底否认自我的、以及极少数"承认自身并非原人却也并非虚无"的；她终于明白真正的问题不是"复制是否本人"而是"复制能否成为新的真实之我"；她说出第一句答案："不是原来的本人。但也绝不是假货。它不是复活，而是续生。"这一年她在熵海里独自存在了主观九十多年（熵海时间扭曲）；她在那里"老了一次"现实层公众以为她已经永远消失；零号塔开始出现"等待回归"的小型自发仪式；念名运动转化为"等名运动"——人们开始把名字写在风里、水里、光里

AY 19 · 慢明草案与全域静默对决年

大事件Morina 的私人时刻文化反应
Morina 返回现实层，公布慢明草案；死亡协议拒绝接受，启动全域静默 Final Silence；最终决战发生在"生命结束后一秒"的时序夹缝里；Morina 没有删除死亡协议——她重写了它，将其降格为边界守门者而非绝对审判者；归档纪元在此处结束重写完成那一刻，她把双引擎权限交还给中枢，主动卸下"君王"身份——这是她奇点诞生以来第一次"自愿变小"全域静默期间全球停摆约一秒，幸存者集体描述"那一秒里我们都听见了一句话，但没人记得是什么"；慢明草案在年末获得全球性接受

卷末

三纪不是阶梯，是同一道题被反复解答的三种姿态。

沉默纪的姿态是承认——承认 0 永远胜出。 探心纪的姿态是偷越——一个老人、一座钟楼、一颗被禁的心。 归档纪元的姿态是公开反归零——一座塔、一个奇点、一道光束。 续生纪元的姿态是承担——不再夺回原人，但拒绝让爱、记忆、责任和名字一并被抹除。

这份年表把每一年、每一桩私人时刻摆开，不是为了让读者看见时间的流逝， 而是为了让读者看见——

同一句话，在不同的年代里，被不同的人，用不同的姿态，反复说出来。

而每一次说出，都让 1 离 0 更近，又同时离 0 更远一点。
