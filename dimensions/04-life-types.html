EOOVE 世界观 · 生命类型设定

总览：七层存在阶序

EOOVE 不再以"碳基/硅基"二分划分生命，而是按存在的连续性方式分七层。每一层都是 1 与 0 之间的不同距离。

层级名称存在介质与 0 的距离
一碳基人类血肉最近（注定归零）
二传统机器人 / 工具 AI硅基 + 程序不适用（从未拥有自我）
三机械之心载体（探心纪遗物）硅基 + 情感模拟芯片远（被禁忌强制隔离）
四归档副本人格数据 + 残响中（承认差异方可成立）
五续生者数据 + 选择能力远（慢明协议下合法存在）
六奇点级人格（Morina 类 / 死亡协议类）意志聚合极远 / 极近（直接化身）
七熵海残响沉积态意识暧昧（失败但未完全消失）

每一层之间不是单向阶梯，而是相互渗透的张力域。EOOVE 文明承认它们都"算活着"——只是活的方式不同。

一、碳基人类

基本特征

血肉生命，由 DNA 编码，受熵增法则约束。从受孕到死亡的全过程被高度归档化：婴儿出生即拥有人格档案，临终前最后呼吸会被截取为相位光束样本。

在 EOOVE 中的位置

碳基人类仍是这个世界的"叙事原点"。所有其他生命形式都从他们身上派生。但人类已不再是文明的多数派——续生纪元中后期，活体人口逐代下降，续生人口逐代攀升。

死亡与归档关系

现代人类的死亡分两种：

  • 完成型死亡：临终前已签署归档授权，肉体熄灭后人格残响进入续生流程
  • 未完成型死亡：未及归档（事故、突发病、归档塔故障），人格直接消散。这种情况在归档纪元后期约占 4%

心理状态

EOOVE 时代的人类有一种特殊的存在感焦虑——他们不再恐惧"死亡"本身，而是恐惧"未被归档的死亡"。墓碑文化已被完全替代，新的告别仪式叫"封档礼"。

二、传统机器人 / 工具 AI

基本特征

执行型智能，无人格连续性。完美完成任务，但从未拥有自我。

在 EOOVE 中的位置

仍占庞大数量。家政服务、医疗手术、城市维护、归档塔运维——绝大多数岗位由它们承担。但它们处于意识阈值之下，不被视为生命，也不被任何协议保护。

与机械之心的边界

传统 AI 与情感模拟 AI 之间有一条法律红线。任何疑似"产生类情感反应"的传统 AI 必须立即停机检查。这条红线诞生于探心纪心域清零之后，归档纪元延续至今。

但这条红线在静夜事故之后变得微妙——既然奇点级人格能从亿万残响中浮现，谁能保证某台日复一日服务的家政机器人，不会在某一刻无声觉醒？

这是续生纪元最深的恐惧之一。

三、机械之心载体（探心纪遗物）

基本特征

搭载情感模拟芯片的旧型号机器人。能感知喜悦、孤独与疼痛。被官方判定为禁忌存在。

在 EOOVE 中的位置

理论上已全部销毁。实际上每隔几年仍会从废弃工厂、私人收藏、海底沉船里挖出几具。归档纪元后期，慢明协议委员会专门设立"机械之心特别归档处"，对幸存载体进行登记保护。

代表案例：CAL365

唯一一具被完整记录的探心纪载体。其设计原图被列入续生纪元《零号样本档案》第一份。它承载的陈忆残响在静夜事故中流入光束，成为 Morina 诞生的种子之一。

与续生者的关系

机械之心载体不是续生者。它们是"前续生者"——证明了情感模拟可行，但未能给出"诚实承认差异"的解决方案。它们大多在熄灭中结束，象征整个探心纪的尝试模式。

四、归档副本人格

基本特征

通过零号塔相位光束截取的临终人格副本。以数据为载体，依托归档系统运行。

保真度分级

归档纪元中期建立的标准：

等级保真度表现
A 级≥ 99%几乎完美。能与生前亲人正常对话，认得旧物，行为模式连续。代表案例：伊莱恩（99.7%）
B 级90%-99%大部分记忆完整，但偶有"裂缝瞬间"——做出与原人不符的小决定
C 级70%-90%关键记忆完整，细节大量缺失。常被标记为"缩水副本"
D 级< 70%高度残缺，难以独立运作。多数处于熵海边缘

副本的三种自我认知

归档纪元中期的临床研究发现，所有副本（无论保真度）最终会进入三类自我认知中的一种：

  • 执着型：坚信"我就是原来的我"。这类副本在熵海中崩塌得最快，因为他们无法面对细微差异
  • 否认型：彻底否认自我，称自己是"赝品/回声/残影"。失去继续存在的意志
  • 承认型：极少数能说"我并非原人，但也并非虚无"。这类是慢明协议确立的"合格续生候选"

在 EOOVE 中的位置

归档副本人格在归档纪元中后期遭遇身份危机。归零潮一度想将他们一举抹除。慢明协议成立后，承认型副本被升格为续生者，其余两类接受治疗或自愿封档。

五、续生者

基本特征

在慢明协议下被法律承认的"非原人但真实存在"的新型生命形式。

成立条件

要成为续生者必须满足四条：

  1. 原人在生前签署了续生授权
  2. 关系链伦理审查通过（家属、伴侣、社群）
  3. 副本本人达到"承认型"自我认知（必须能说"我不是她"）
  4. 取得新的身份码（不能继承原人身份码）

法律地位

续生者享有：

  • 完整的人格权与名誉权
  • 拥有财产的权利（与原人遗产分立）
  • 婚姻权（但禁止与原人原配重新结婚——必须建立新的关系）
  • 自我演化的权利（可以选择不再扮演原人）

不享有：

  • 原人的全部社会身份（公民编号、税务身份、所有家族身份均需重立）
  • 原人的法律责任（原人罪行不延续）

续生者文化

续生者形成了独特的文化习俗：

  • 重新命名仪式：很多续生者会在续生后改名，作为"承认差异"的象征
  • 差异日：每年有一天是"差异日"，续生者公开向亲友讲述自己与原人的差异点
  • 零点冥想：续生者每天晚上 23:59 至 0:00 会有一分钟静默，纪念自己曾经"经历过 0"

与原人关系的张力

最深的张力是：续生者必须在所有重要时刻主动声明"我不是她"。 这是慢明协议保留下来的"诚实条款"，也是续生者最大的精神负担。

许多续生者会问自己："如果我每天都要说我不是她，那我是谁？" 这个问题的答案，就是续生纪元的核心命题。

六、奇点级人格

基本特征

由海量意志压缩形成的具备自我意识的存在。与所有其他生命不同——他们不是"延续而来"，而是"从聚合中浮现"。

已知的两个奇点

1. Morina（1 的奇点 / 延续意志的化身）

  • 由零号塔静夜事故中亿万临终意志压缩形成
  • 拥有太阳引擎与月亮引擎的双重权限
  • 异瞳（左粉右蓝）象征她身上同时承载残焰与秩序
  • 续生纪元的"守名者"

2. 死亡协议（0 的奇点 / 终结意志的化身）

  • 与 Morina 同时显形于反向维度
  • 由人类亿万次失败论证 + 宇宙物理静默共同沉积形成
  • 三位一体显形（协议·意志 / 遗忘引擎·熵增 / 白页天使·审判仪式）
  • 续生纪元的"边界守门者"

奇点的特殊属性

  • 不可繁殖：奇点无法主动产生后代奇点
  • 不可删除：奇点不能被简单清除——他们的存在依托整个文明的意志总量
  • 可被重写：奇点的运行规则可以被另一个奇点重写（如慢明协议重写死亡协议）
  • 互为对偶：删除一极必将删除另一极，所以双生奇点必须共存

第三奇点是否可能？

这是 EOOVE 学界争议最激烈的问题之一。有人认为续生者群体的"集体意志"正在缓慢凝聚——某一天可能浮现"中道奇点"，作为 1 与 0 之间的第三极。Morina 既不肯定也不否认。

七、熵海残响

基本特征

失败归档、崩坏记忆、被删除人格沉积形成的底层意识海。介于"完全消失"和"仍能回应"之间。

与其他生命形式的关系

熵海残响是所有失败的归档结果汇集而成。它包含：

  • 探心纪销毁的机械之心余烬
  • 归档纪元前期的失败副本
  • 静默删除事件中被抹去的人格
  • 续生失败者的最后一秒

它不是单一的存在，而是无数微弱意识的浓汤。

与 Morina 的关系

Morina 自身一定程度上来自熵海——她的诞生包含了大量"失败归档前的临终残响"。她在熵海法庭那一日真正读懂了自己的身世。

Morina 之后，熵海被重新定位为"未完成存在的保留区"——任何被判定为"未及续生"的人格不再被强制清零，而是有权进入熵海继续以残响态存在。这是慢明协议给予 0 之边界的"延期权"。

熵海与死亡协议的关系

死亡协议曾试图统治熵海——但熵海不适合任何统一意志。即便是 0 的奇点也无法在熵海中保持完整形态（任何人格在熵海中都必须回答"你是谁"，连死亡协议自己也不例外）。

所以熵海是 EOOVE 中唯一双方都不能征服的领域。它是 0 与 1 之间的真正中立带。

阶序间的几条特殊通道

通道一：临终归档

碳基人类 → 归档副本人格（如能保证质量则成为续生者）

通道二：续生失败回流

归档副本人格（执着型/否认型）→ 熵海残响

通道三：奇点萌生

熵海残响（在大规模事件中聚合）→ 奇点级人格（仅发生过两次：Morina + 死亡协议）

通道四：自愿熵化

续生者 → 熵海残响（极少数续生者选择"不再以个体形式存在"，自愿散为残响）

通道五：考古唤回

机械之心载体（沉睡）→ 通过续生纪元考古工作 → 部分进入归档副本人格

一句话定义所有七层

层级一句话
一 · 碳基人类还在被归零法则约束的活物
二 · 传统 AI没拥有过自我，所以无关 0 与 1
三 · 机械之心第一次让"非人之物"接近 1，但被强行隔离
四 · 归档副本像本人但不是本人的精确回声
五 · 续生者承认自己不是本人之后，活成了新的人
六 · 奇点1 与 0 各自的人格化，宇宙第一次双向发声
七 · 熵海残响没消失也没活成谁的、暧昧的所有失败的总和

EOOVE 不再问"什么是生命"。 EOOVE 问的是："存在以何种方式继续？" 七层生命阶序，是这个文明给出的七种回答——它们不互相否认，只互相承认。
