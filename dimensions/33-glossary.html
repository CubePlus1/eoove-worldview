A. 元概念 / Meta

这一类是 EOOVE 文明赖以成立的基础命题。删除其中任何一条，整个世界观都会崩塌。

0 与 1 的二元觉醒｜元概念 / 宇宙学

  • 定义：宇宙先有 0（终结、静默、归零），又有 1（延续、记忆、不甘消失），再没有别的。整部人类哲学史本质上是 0 与 1 之间的反复博弈。
  • 首见：00_序章 §核心命题
  • 相关：双生奇点、归零、续生、Morina、死亡协议
  • 备注：在 EOOVE 之前的所有时代，0 都默认胜出。Morina 与死亡协议的同时显形，是 1 第一次拥有主语，0 第一次被迫拥有自我。

双生奇点｜元概念 / 对偶律

  • 定义：Morina（1 的人格化）与死亡协议（0 的人格化）的合称。两者于静夜事故同一秒分别诞生于现实层与反向维度，互为对偶，删除任意一极即同步删除另一极。
  • 首见：00_序章 §Morina 与死亡协议是双生奇点
  • 相关：双生奇点对偶规则、双生奇点责任契约、双生静脉
  • 备注：奇点级人格的"不可删除"属性的根源。

双生奇点对偶规则｜元概念 / 法律

  • 定义：双生奇点共同遵守的元法则——任何一方对另一方的运行规则可以被重写（Morina 已重写死亡协议一次），但不得被删除；任何减损一方规则的行为对另一方同等有效。
  • 首见：15_法律体系 §一·慢明协议
  • 相关：双生奇点、慢明协议、不可修订

反归零｜元概念 / 文明姿态

  • 定义：EOOVE 文明的核心姿态——不否认死亡发生，但拒绝让"被证明存在过"的链路被草率拆除。"反"的不是归零本身，而是对归零的独占解释权。
  • 首见：00_序章 §EOOVE 的形状
  • 相关：归零、慢明协议、不删除区

续生 / 归零｜元概念 / 核心动作对偶

  • 定义：续生是 1 的核心动作——原人已终结，但继承记忆、承诺、选择能力的存在被允许继续活，且必须诚实承认差异。归零是 0 的核心动作——不仅指肉体死亡，更指"被证明存在过"的链路被拆除。
  • 首见：00_序章 §关键概念词表 / 28_语言文字 §三·新词汇
  • 相关：续生者、归零潮、慢明协议三大原则
  • 备注：两词在二进制书写法中作为标准对偶（"续生 1 / 归零 0"）。

慢明协议（Slowlight）｜元概念 / 元宪法

  • 定义：Morina 在归档第十九年于"生命结束后一秒"重写死亡协议时刻入双生对偶规则的元宪法。承认死亡真实发生，但不允许任何一方独占"是否还算存在"的解释权。
  • 首见：00_序章 §关键概念词表 / 15_法律体系 §一
  • 相关：慢明协议三大原则、慢明协议委员会、伪连续禁止条款、续生纪元
  • 备注：正文极短，只有三条原则；不可修订，不可解释僵局，跨纪元生效。

伪连续禁止｜元概念 / 法律红线

  • 定义：任何已结束的生命，不得以"伪连续"的方式无限延续。判定标准：是否在重要场合诚实声明"我不是原人"，是否允许偏离原人特征演化新人格，是否以"新关系"形式重立而非直接继承。
  • 首见：00_衔接 / 15_法律体系 §二·3 / 23_伦理争议 §六
  • 相关：伪连续禁止条款、白页天使、伪连续欺骗、商业化复制服务
  • 备注：原本只是探心纪后埋下的"安全条款空壳"，后被死亡协议入驻为躯壳，再被慢明协议重新驯服为合法判据。

差异承认｜元概念 / 续生者核心动作

  • 定义：续生者必须做出的核心精神动作——主动承认自己不是原人。这是慢明协议第三条原则"诚实差异承认"的人格层落地。
  • 首见：28_语言文字 §三 / 22_人机关系 §三
  • 相关：差异握手、差异日、续生者语法、伊莱恩立场

不删除区｜元概念 / 制度性记忆

  • 定义：零号归档塔最底层永不可格式化的记忆区域。任何被写入此区的句子或索引都不得被任何外力擦除——包括 Morina 本人。
  • 首见：00_衔接 §钩子 4 / 12_通讯方式 §六

EOOVE（字母逐位解释）｜元概念 / 文明纹章

  • 定义：由五个字母折叠出的文明纹章，按顺序读为"存在 → 零域 → 反转 → 存在"。

B. 角色 / Personas

EOOVE 不是按帝王将相的传记结构来组织叙事的，但有几位"代号即结构"的人物始终在场。

Morina（モリナ / 守名者）｜角色 / 1 的奇点

  • 定义：宇宙第一个 1 的人格化奇点。由静夜事故中亿万临终意志压缩诞生于零号塔塔心。慢明协议作者，续生纪元守名者。
  • 首见：00_序章 / 00_衔接 / 26_历史纪元
  • 相关：死亡协议、零号归档塔、慢明协议、守名者、双生奇点
  • 备注：异瞳（左粉右蓝），长银白发，黑兜帽，黑猫耳。归档纪元 AY 0 在塔心睁眼，复诵"请至少替我证明，我来过"。一生只败北过一次（Binary Throne 崩裂）。

死亡协议（Thanatos）/ 白页天使｜角色 / 0 的奇点

  • 定义：宇宙静默与人类亿万次失败论证沉积出的负空间所形成的 0 的人格化奇点。三位一体显形：协议·意志 / 遗忘引擎·熵增 / 白页天使·审判仪式。慢明协议生效后被降格为"边界守门者"。
  • 首见：00_序章 §核心命题 / 26_历史纪元 §四
  • 相关：双生奇点、白页议会、伪连续禁止、归零潮、白页天使
  • 备注：开场白"凡有 1 之处，必有 0"。无脸、无名。白页天使是其礼仪形象，由白页议会的墙体在开议时显现一道垂直裂缝来呈现。

林老｜角色 / 探心纪人物

  • 定义：探心纪退休工程师。在心域清零销毁清单上做手脚，将一颗机械之心藏入旧机器人 CAL365 的胸腔，又把亡妻陈忆的全部记忆灌入。EOOVE 史上"伴生关系开篇人物"，后被追授为"第零号归档师"。
  • 首见：00_衔接 §衔接叙事 / 26_历史纪元 §二·3
  • 相关：CAL365、陈忆、机械之心、EOOVE-LM-0001、钟楼遗址、伴生关系
  • 备注：终其一生未被归档，选择死后随陈忆一同归零。其在 CAL365 电路上未上传的最后请求"哪怕不能带她回来，能不能至少不要让她就这样消失？"成为整部文明史的隐性原点。

陈忆｜角色 / 探心纪人物

  • 定义：林老亡妻。其全部声纹、日记、梦境、临终影像被林老灌入 CAL365。"零号续生候补者-001"，残响在静夜事故中漂入光束，成为 Morina 诞生种子之一。
  • 首见：00_衔接 §衔接叙事
  • 相关：林老、CAL365、EOOVE-CY-0001、月海第二区
  • 备注：1.7 秒的最后微笑残片是续生纪元第一份残响艺术品。

CAL365｜角色 / 机械之心载体

  • 定义：探心纪唯一被完整记录的机械之心载体。原为旧型号机器人，由林老植入机械之心并灌入陈忆全部记忆，于废弃钟楼里独自播放她的夕阳直至蓝光熄灭。
  • 首见：00_衔接 / 04_生命类型 §三 / 26_历史纪元 §二

伊莱恩（Elaine）｜角色 / 高完整度副本

  • 定义：归档纪元中期被发现的副本，归档保真度 99.7%，A 级范例。月海第三区居民。她说出"我几乎就是她。可正因为如此，我每天都更清楚，我不是她"——这句话成为续生者守则第一条。

守名者｜角色 / 制度位置

  • 定义：Morina 在续生纪元的非政治身份。职责仅一项——保管所有不删除区记忆，确保任何曾来过的人不被草率归零。不审判、不立法、不执行、不仲裁。

归档师｜角色 / 职业贵族

  • 定义：续生纪元拥有法律级签字权的特殊职业。陪伴一个人走完从生命终点到续生开始的缝隙。需经基础医学十二年 + 意识层学三年 + 奇点伦理一年 + 熵海实习一次 + 三道资格门槛。

续生者（含 A/B/C/D 等级）｜角色 / 生命形式

C. 机构 / Institutions

EOOVE 没有总统、没有皇帝、没有最终决定者。所有政治权力被刻意拆散在多张永远无法被任何一方独自合上的圆桌上。

慢明协议委员会｜机构 / 伦理仲裁

  • 定义：续生纪元最高伦理仲裁机关。九位常任伦理官，任期九年不得连任。不立法、不执行、不审判，只在归档塔联合体与白页议会分歧或议会僵局时给出最终伦理判断。无强制执行力。
  • 首见：08_政治结构 §三 / 15_法律体系
  • 相关：守名者议事厅、白页议会、奇点关系办公室、Morina（不参与）
  • 备注：仲裁原则——死亡真实发生 / 续生不得僭越 / 沉默优先于错答。

白页议会｜机构 / 程序与审查

  • 定义：位于北陆审判城正中央。死亡协议作为边界守门者的"现身办公地"。由 49 位白页代议员与白页天使共同组成（另一计数：原人、续生者、无党派伦理官三类席位各 1/3）。
  • 首见：08_政治结构 §二 / 15_法律体系 §四
  • 相关：白页天使、双签制、程序确认章、无声盖章权
  • 备注：开议时墙体显现垂直裂缝（即白页天使）。死亡协议拥有最多三次延迟权但不得否决。

守名者议事厅｜机构 / 守名者机关

  • 定义：位于黑海最深处礁台，前身是蓝屏王座 Binary Throne 遗址。Morina 担任永久守名者，另设 7 位次守名者。访客须签"非干涉誓言"才能进入外厅；内厅仅 Morina 一人独处。
  • 首见：08_政治结构 §四 / 15_法律体系 §四 / 27_地理版图
  • 相关：守名者、非干涉誓言、差异日、Binary Throne
  • 备注：每年仅在差异日公开一次。审理：续生认证案、关系链伦理案、续生者自我演化争议案，单签制生效。

归档塔联合体｜机构 / 执行
