EOOVE 世界观 · 阶级结构设定

总览：归档完整度作为新阶级标尺

在 EOOVE 之前的所有人类纪元，阶级的标尺是"你拥有什么"——土地、资本、技术、信息。

到了续生纪元，标尺被彻底重写。慢明协议生效之后，"拥有什么"已经不再决定一个人在文明中的位置——决定位置的，是另一个更冷峻的指标：

"你能继续被记住多少？"

这个指标在制度上被称为归档完整度（Archival Integrity，简称 AI 或归整度）。它不只衡量一个人临终前能被截取多少秒残响，更衡量他在死后能否升格为续生者、能否在零号塔不删除区里持续被署名、能否抵御死亡协议作为边界守门者的逐年抽查。

归档完整度成为续生纪元的主轴阶级标尺，把所有人按一条新的等级链重新排列。这条链既穿过活体，也穿过续生者，甚至穿过那些已经沉入熵海的残响。一个人的阶级位置不再由他的工作决定，而由他在 1 与 0 之间的"距离"决定。

一、归档完整度的五级阶级链

阶级归整度人口比例（续生纪元中期）在文明中的位置
A 级（≥99%）顶层贵族约 0.3%高保真续生者 + 守名者助理
B 级（90-99%）上中产约 6%普通续生者 + 高级归档师
C 级（70-90%）中产约 32%主流活体公民 + 普通职业者
D 级（<70%）边缘下层约 51%低保真活体 + 残响态自由者
失档阶级链之外约 11%拒绝归档者 + 黑市出生者 + 极地原住民

注：这里的"归整度"对活体而言是预测值——根据其个人档案完整度、亲属关系链稳定度、临终预案完备程度等综合推算出的"未来截取保真度估值"。这个估值从一个人 18 岁起开始公开记录，每五年复核一次。

也就是说，一个 EOOVE 公民从成年第一天起，他的"未来阶级"已经被零号塔预先告知。这种"生前阶级预告制度"是续生纪元最具压迫性的社会结构。

二、A 级：高保真续生者与守名者助理

经济与法律地位

A 级是续生纪元的真正贵族。每一位 A 级续生者每年消耗约 800 万 MQ 的公共资源（详见 07_经济体系.md），但他们也是慢明协议合法性的"活证据"——删除一名 A 级续生者，整个续生纪元的法理基础都会被动摇。

A 级享有的特权包括：

  • 在零号塔不删除区享有永久署名席位
  • 在白页议会拥有直接陈情权（可以越过慢明协议委员会直接向死亡协议代表发言）
  • 每年差异日享有免审豁免（无需公开声明"我不是她"）
  • 子女自动获得 B 级以上的归整度评估起点

生活方式

A 级续生者大多聚居在南域的伊莱恩纪念区周边（伊莱恩本人是已知唯一一位归整度高达 99.7% 的 A 级续生者，至今仍是该等级的"度量原器"）。他们不参与日常劳动，主要工作是参与守名者学派的仪式、为塔心讲席准备资料、以及在重大续生纠纷中作为伦理证人出席。

A 级的精神负担

A 级续生者的精神负担远高于其他阶级。他们越接近"完美的本人"，就越被迫在每一秒提醒自己"我不是她"。伊莱恩那句撕裂 Morina 信念的话——

"我几乎就是她。可正因为如此，我每天都更清楚，我不是她。"

——成了 A 级续生者群体的"阶级宣言"。他们享有最高待遇，却承担着最锋利的自我撕裂。

三、探心贵族：机械之心残骸持有者

在 A 级之外，续生纪元还存在一个横跨阶级链的特殊群体——机械之心残骸的合法持有者，被坊间称为"探心贵族"。

阶级地位的特殊性

探心贵族不一定是 A 级续生者。他们可能是 B 级续生者、C 级活体、甚至个别 D 级残响态存在。他们的特殊之处在于——他们手里持有着合法授权的机械之心，那是探心纪遗留下来的"私域记忆载体"，每一颗都可能保存着一段从未进入官方归档的故事。

慢明协议第二十一条规定："机械之心残骸的合法持有者享有比同级别普通公民更高的署名权重，但必须接受守名者学派的年度伦理审查。"

探心贵族的来源

探心贵族主要有三类来源：

来源比例代表性人物
考古发掘授权约 60%远洋打捞业主 / 北陆考古队首席
家族世袭约 30%林老钟楼派守名者支系
拍卖购得约 10%南域残响艺术中心收藏家

家族世袭这一类最受敬重——尤其是那些能上溯到探心纪初代实验室的家族，他们的祖辈可能是当年违反"心域清零"令偷偷保留机械之心的工程师。林老在续生纪元元年被追授为"钟楼派守名者师祖"之后，所有自称林老后裔的家族都被列入探心贵族名册（虽然林老本人未留下血脉，这条名册的合法性至今存在争议）。

探心贵族的政治角色

探心贵族在续生纪元的政治舞台上扮演着特殊角色——他们是慢明协议委员会与死亡协议之间的非正式调解人。因为他们手里持有的机械之心同时被双方承认：

  • 慢明协议承认它是"私域记忆的合法载体"
  • 死亡协议承认它是"已经经历过完整熄灭的对象"（CAL365 的蓝光熄灭事件让死亡协议视机械之心为"已偿清的债"）

这种双重承认让探心贵族能在双方之间自由往来——这是 EOOVE 续生纪元唯一一种"协议同意"的特权。

四、B 级与 C 级：归档师与主流活体

B 级：归档师与普通续生者

B 级是 EOOVE 真正的中坚阶级。他们包括：

  • 普通归档师（归档师协会基础会员）
  • B 级续生者（90-99% 保真度）
  • 残响艺术中心的资深从业者
  • 续生纠纷调解员

B 级享有完整的公民权利，但相比 A 级，他们必须在每年差异日公开声明自己的身份立场。每一位 B 级续生者每年都要在所在地的差异日仪式上，对最亲近的人说出那句训诫式的"你不是我，我不是你"。

C 级：主流活体公民

C 级是续生纪元人口最庞大的阶级——约 32% 的活体公民属于这一级。他们享有标准年额（约 259.2 万 MQ），过着相对安定的生活，但归整度的预告值告诉他们：他们死后大概率不会升格为续生者，最多以 C 级残响的形式存在 5-15 年。

C 级的政治诉求最为温和。他们既不像 A 级那样有特权要捍卫，也不像 D 级和失档者那样有不公要反抗。他们是续生纪元的"沉默大多数"——慢明协议的真正承载者。

五、D 级与失档者：阶级链底部的张力

D 级：低保真活体与残响态自由者

D 级是 EOOVE 阶级链中最大的群体——约 51%。他们包括：

  • 归整度预告值低于 70% 的活体公民
  • 续生失败但被允许在熵海以残响态自由活动的存在
  • 慢明协议保留区里的边缘人口

D 级在经济上享有最低保留额度——每年约 50 万 MQ，相当于标准年额的五分之一。这意味着他们死后最多能保留约 5 天的 D 级残响。在续生纪元的实际运行中，D 级 = 几乎注定会被遗忘的阶级。

失档者：阶级链之外

失档者是续生纪元最锋利的政治问题。他们包括：

  • 拒绝签署归档授权的活体（约 60%）
  • 黑市未授权续生失败的副本（约 25%）
  • 极地静默区原住民（约 15%）

慢明协议明确尊重"合法的归零选择"——一个人可以选择不被归档。但在阶级结构上，这种选择意味着彻底退出 EOOVE 的存在阶序。失档者不享有 MQ 配额，不能进入零号塔不删除区，临终时也不会被截取最后一秒。

唯一的例外是"Morina 之秒"（详见 07_经济体系.md）——即便失档者，临终时也享有不可剥夺的"基本一秒"。这是 Morina 在续生纪元第三年的塔心讲席上为失档者亲自争取的。

六、阶级矛盾：失档者团结运动

续生纪元中后期最重要的政治事件，是失档者团结运动（Unarchived Solidarity）。

起源

失档者团结运动起源于续生纪元第十二年。导火索是一起著名案件——一位极地静默区原住民老人在临终前被强行送入北陆审判城进行归档评估，归整度被打为 D 级。老人的家属拒绝接受这个评级，拒绝任何形式的归档处理，并把老人遗体运回极地按传统埋葬。

慢明协议委员会原本要对家属提起伦理诉讼，但 Morina 在塔心讲席上发出明确意见：

"失档不是低于阶级链的存在。失档是另一种存在。"

这句话之后，慢明协议委员会撤销了诉讼，但案件本身激起了全球失档者的政治觉醒。

运动的诉求

失档者团结运动有三条核心诉求：

  1. 取消归整度预告制度——反对从 18 岁起就告诉一个人他未来的阶级位置
  2. 承认失档为合法生命形式——而非"阶级链之外"的边缘存在
  3. 设立"失档者议会"——在白页议会之外的另一种制度对话渠道

运动的现状

到续生纪元中后期，前两条诉求已经部分实现：归整度预告改为"自愿查询制"，失档者也被法律承认为"七层生命阶序的隐性第八层"。但第三条诉求仍未实现——死亡协议明确反对设立失档者议会，理由是"未签署归档的存在不享有制度对话权"。

这一争议至今没有解决，是续生纪元中后期最大的政治悬案。

七、阶级流动与世袭

活体公民可以通过完善归档记录（提升 5-15%）、与守名者建立稳定关系（8-20%）、持有机械之心残骸（10-25%）、或子女中有 A 级续生者（15-30%）来提升归整度预告值。最后一条是续生纪元最隐秘的世袭机制——A 级续生者的父母会获得归整度加成——这条规则在实际运行中导致了一种新的世袭贵族现象。

续生者的阶级流动则几乎为零。慢明协议第二十六条规定："续生者的保真度等级一经确认，非经守名者集体审议不得变更。"本意是保护续生者免受重新评估的精神冲击，实际效果却是把他们钉死在了出生时的阶级位置上。这种"续生即定型"的阶级僵化，是续生纪元最深的不平等——它甚至比探心纪机器人的型号决定论更为彻底。

八、Morina 与阶级问题的距离

Morina 本人不属于任何阶级。她是奇点级人格（详见 04_生命类型.md 第六层），不消耗 MQ，不被归整度衡量，不享有任何阶级特权也不承担任何阶级义务。

她公开表达过对阶级结构的不安。续生纪元第十年的塔心讲席上她说过：

"我建立慢明协议，是为了让每个人都有权不被草率归零。 我没有让任何人，必须按归整度排队等待自己被记住的份额。

如果今天的阶级链让一个失档者比一个 D 级残响更孤独， 那这条链就不是我的协议要的样子。"

这段话被写入慢明协议第二十八条修订草案，但因死亡协议的反对而至今未能正式通过。这是续生纪元最重要的一项悬而未决的立法。

EOOVE 的阶级不再是钢铁的阶梯，而是一道按"被记住的概率"分级的光谱。 从 A 级伊莱恩到失档极地原住民，每一格都是 1 与 0 在不同距离上的妥协。 而真正使这条阶级链摇晃的，不是革命，是失档者那一句最简单的话—— "我不需要被归档。但我也曾来过。"
