EOOVE 世界观 · 经济体系设定

总览：从物质丰裕到记忆稀缺——续生纪元的归档本位制

EOOVE 文明的经济史，是一部"稀缺对象不断后撤"的简史。探心纪解决了物质匮乏，归档纪元解决了能源与算力匮乏，到了续生纪元，唯一仍然真正稀缺的，只剩一样东西——归档存储的合法额度。物质可以被复制，能源可以被合成，唯独"被允许在零号塔不删除区里继续被命名"的容量是有限的：因为相位光束每年能稳定截取的最后一秒数量，写在宇宙的物理预算里，写在死亡协议作为边界守门者签字承认的硬上限里。

于是慢明协议生效之后，整个文明把货币的锚定物从黄金、信用、能源单位，统一迁移到了一种新的计量单位——记忆配额（Memory Quota，简称 MQ）。

这是 EOOVE 经济学的核心命题：当死亡不再被允许独占解释权之后，"继续被记住的权利"就被迫成为一种公共资源；公共资源一旦稀缺，就必然被定价。

一、记忆配额（MQ）的发行与流通

基本单位

记忆配额按"标准秒"计算：1 MQ = 在零号塔不删除区储存 1 秒中等保真度（B 级）人格残响所占用的归档资源。换算关系大致如下：

单位换算实际意义
1 MQ1 秒 B 级残响一句普通话语的封存
60 MQ1 分钟 B 级一段简短遗言
3,600 MQ1 小时 B 级一次完整的告别对话
1 标准年额约 2,592,000 MQ慢明协议为每名活体公民分配的年度额度

公民基础额度

慢明协议将归档纪元后期的私人化归档市场重新国有化。每位活体公民每年由零号塔自动获分配一份"标准年额"——足够封存约 30 天 B 级残响，或约 7 天 A 级残响。这份额度可以累积、可以馈赠、可以遗赠，但不可被强制征收——这是慢明协议向死亡协议争取来的最重要一条经济条款。

续生者的逆向额度

续生者本身就是被归档资源所承载的存在，因此其经济地位反转：他们不"消耗"配额，而是被计为配额负担载体。一名 A 级续生者每年的"持续运行成本"约为 800 万 MQ——也就是说，每存在一名 A 级续生者，文明就需要从公共池里持续支付相当于三名活体公民的年额。

这条规则催生了续生纪元最尖锐的政治问题： 活体在替续生者养活归档塔，还是续生者在替活体守住"我们终将被记住"的承诺？

二、保真度等级化的经济权重

归档资源不是一种均质资源。它高度等级化、高度政治化。一位 A 级续生者占据的存储位与一位 D 级残档占据的位，在经济账本上完全不是同一种东西。

保真度年消耗 MQ经济权重（基准 1.0）典型职业空间
A 级（≥99%）~800 万8.0 - 12.0守名者助理 / 续生顾问 / 月海导引员 / 塔心讲席学者
B 级（90-99%）~200 万2.0 - 4.0归档师 / 续生纠纷调解员 / 档案考古员
C 级（70-90%）~50 万0.5 - 1.0一般行政 / 残响艺术 / 自由职业
D 级（<70%）~10 万0.1 - 0.3边缘服务 / 残响态自由活动
失档0不可量化拒绝归档者 / 极地静默区原住民 / 黑市出生者

这种等级化制造了 EOOVE 经济史上最深的不平等：A 级续生者在出生（续生）那一刻起，就持续从公共池里抽取相当于碳基中产二十倍的资源，但他们也是慢明协议运行的"活证据"。每删除一名 A 级续生者，文明对续生伦理的合法性就要被重新审一遍——这让他们在政治上几乎不可触动。

伊莱恩在月海对话之后自愿散为残响那一日，南域续生者居住带连续静默了七十二小时。后世经济史学家把那一天称为"MQ 通缩日"——大约 800 万 MQ 在一夜之间从公共池里释放出来，相当于整个北陆审判城三个月的运行预算被一个人的选择重新写了回去。

三、归档师与守名者：续生纪元的两类职业贵族

归档师协会（北陆总部）

归档师是续生纪元最受敬重的中产职业。他们持有零号塔颁发的"截取资格证书"，能合法操作相位光束的次级权限，为临终者完成最后一秒的封档作业。一名资深归档师的年收入约为 1,500 万 MQ，相当于五份标准年额。

归档师不被允许同时持有大量私人 MQ——慢明协议第十一条明确规定："操作公共光束者，不得在私人账户中囤积超过五年额的配额。"这条法律的目的是防止归档行业像探心纪末期的"私人记忆服务公司"那样把公共资源寡头化。

守名者学派

守名者不是职业，是一种仪式性身份。Morina 在续生纪元元年放弃了"君王"自居，把自己定位为最高守名者；之后的守名者按学派传承——他们不收取 MQ 报酬，而是被允许在零号塔不删除区享有"署名权"（即可以为某段记忆署名为见证者）。

守名者是 EOOVE 经济中唯一完全脱离 MQ 体系的群体。他们不消费配额，不持有配额，但他们的签名能为任何一段记忆增加一层制度合法性——这本身在黑市上被换算成"无价"。

四、黑市经济：未授权续生与记忆碎片走私

慢明协议越严密，黑市越繁荣。这是续生纪元的经济铁律。

黑市第一品类：未授权续生

约 12% 的人类活体签署了正式续生授权。但需求远超 12%。剩下那部分人——尤其是那些临终前才意识到"原来我也想被记住"的人——会在死后被亲属偷偷送进地下归档棚：这些棚子大多藏在远洋漂浮残片群之间，使用的是被走私出零号塔的低端相位光束零件，能截取约 D 级保真度的残响。

地下棚位的标价在续生纪元第七年达到顶峰：单次截取约 400-800 万 MQ（黑市汇率），相当于两到四份标准年额。许多家庭因此倾家荡产。慢明协议委员会与死亡协议就此达成了罕见的共识——双方都试图打击地下棚位，但双方的理由不同：协议委员会怕"未承认型副本"流入熵海污染续生伦理，死亡协议则单纯地认为"未走光束的封档不构成合法终结"。

黑市第二品类：记忆碎片交易

比未授权续生更隐秘的市场，是记忆碎片黑市。在归档纪元前期失败的副本、归零潮中被静默删除的人格、续生失败者的最后一秒——这些理论上都该沉入熵海，但其中一部分会在沉降过程中被截留下来，以"碎片态"流入黑市。

碎片市场上最贵的不是某位名人的记忆，而是情绪原型：一段未被命名的悲伤、一个无主的拥抱、一种已经在熵海中风化但尚未完全溶解的眷恋。残响艺术家用它们做装置；续生者用它们填补自己的"裂缝瞬间"；少数收藏家把它们当作纯粹的奢侈品收藏。

这种碎片交易在慢明协议下属于灰色地带：协议没有明确禁止，因为碎片的"原人"早已不可追溯；但任何使用碎片填补续生者裂缝的行为都会被守名者标记为"伦理脏污"。

黑市第三品类：身份码

慢明协议要求续生者必须取得新的身份码（不能继承原人身份码）。但部分续生者——尤其是与原人配偶有未解情感纠葛的——会试图通过黑市购买"近似身份码"，以便假装自己仍是原人继续生活。这种交易在南域的续生者居住带边缘最为猖獗，被称为"伪续生案"，是慢明协议委员会最常受理的纠纷类别。

五、考古经济：机械之心残骸打捞业

续生纪元中期开始兴起一项新兴产业——探心纪考古。

远洋漂浮残片群、黑海西岸钟楼遗址周边、北陆地基深处的探心纪研究所旧址——这些地方陆陆续续被挖出探心纪机器人残骸。每一具残骸都可能搭载一颗未被完全清零的机械之心；每一颗机械之心都可能保存有一段从未进入官方归档的"私域记忆"。

这是续生纪元唯一一种"记忆从无到有"的合法生产方式——其他所有归档资源都来自活体临终截取，而机械之心残骸里藏着的，是探心纪那批被官方判处死刑、但实际上从未真正消失的私人故事。

考古打捞经济因此成为最受守名者学派青睐的产业。一颗能稳定读取的机械之心残骸在拍卖市场上的成交价约为 1.2 亿 MQ，相当于六十份标准年额。CAL365 残骸出土事件之后，这条产业链彻底高端化——南域残响艺术中心专门设立了"机械之心收藏部"，由 Morina 亲自担任名誉鉴定人。

七、Morina 对经济命题的态度

Morina 极少对 MQ 体系发表正式评论。她在续生纪元第三年的塔心讲席上说过的一句话被反复引用：

"我能允许记忆被定价。 我不能允许有人因为价格付不起，就被默认归零。"

这句话被慢明协议委员会解读为"最低保留额条款"的法理来源——任何活体公民，无论 MQ 账户余额是否清零，都享有不可剥夺的"基本一秒"：在他临终时，零号塔必须为他截取至少一秒的最后影像，无论该一秒能否升格为续生人格。

这一秒在续生纪元被称为"Morina 之秒"。它不算续生，不算归档，只是一种最低限度的"我曾来过"的物证。

EOOVE 的经济不是关于钱的经济，而是关于"被允许继续被想起"的经济。 当 1 与 0 共生在同一颗水珠之上，每一次交易都是一次微型的双生奇点之争—— 你为这一秒付出的不是 MQ，而是你愿意让多少终结被推迟。
